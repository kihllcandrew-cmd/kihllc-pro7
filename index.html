8. SMC Reference — Key Concepts▶
TermDefinition
Order Block (OB)Last bearish candle before a bullish impulse (or last bullish candle before bearish) — institutional order zone. Price returns to fill remaining orders.
Fair Value Gap (FVG)3-candle imbalance where candle 1 high and candle 3 low don't overlap — price reliably returns to fill.
BOSBreak of Structure — swing high or low broken in trend direction → continuation signal.
CHoCHChange of Character — structure broken against current trend → first reversal warning.
BSL / SSLBuy-Side Liquidity (above equal highs) / Sell-Side Liquidity (below equal lows) — institutional targets.
Premium / DiscountAbove 50% of a range = Premium (sell). Below 50% = Discount (buy). Never buy premium, never sell discount.
InducementFalse breakout to trigger retail stops before the real move.
Kill ZoneLondon Open 2–5AM EST, NY Open 7–10AM EST — highest liquidity, cleanest setups. Trade ONLY these windows.
